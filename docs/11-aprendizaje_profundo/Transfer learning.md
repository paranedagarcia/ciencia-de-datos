# Transfer Learning
Imagina que quieres aprender a conducir un camión. ¿Tendrías que aprender desde cero qué es un volante, para qué sirve el freno o cómo mirar por los espejos? ¡Claro que no! Como ya sabes conducir un coche, transfieres ese conocimiento al nuevo reto.

En Inteligencia Artificial, el Transfer Learning es exactamente eso. En lugar de entrenar una red neuronal desde cero (lo cual requiere miles de imágenes y semanas de computación), tomamos un modelo que ya es "inteligente" —porque ha sido entrenado con millones de datos (como el famoso dataset ImageNet)— y lo usamos como base para nuestra tarea específica.

¿Por qué es asombroso?

Ahorras tiempo: No necesitas semanas de entrenamiento.

Ahorras datos: Puedes crear un modelo excelente con solo 100 imágenes en lugar de 100,000.

Eficiencia: Es como heredar una enciclopedia en lugar de tener que escribir cada página tú mismo.
