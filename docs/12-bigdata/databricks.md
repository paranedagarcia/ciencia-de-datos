# Databricks

Databricks es una plataforma de datos e IA en la nube basada en Apache Spark que unifica ingeniería de datos, SQL, ciencia de datos y aprendizaje automático (ML) mediante una arquitectura "Data Lakehouse". Permite gestionar grandes volúmenes de datos, facilitando la colaboración, el procesamiento eficiente y la gobernanza segura en la nube. 

## Características Principales y Componentes
- **Data Lakehouse**: Combina la flexibilidad de los data lakes con la gestión de datos de los data warehouses (transacciones ACID).

- **Apache Spark**: Motor principal de procesamiento de datos distribuido, diseñado para velocidad y escala.

- **Unity Catalog**: Ofrece gobernanza unificada para datos, analítica e IA.

- **Databricks SQL**: Permite ejecutar consultas SQL sobre el Lakehouse con alto rendimiento.

- **Notebooks y Colaboración**: Entorno interactivo para científicos e ingenieros de datos.

- **Delta Live Tables**: Facilita la creación de pipelines ETL confiables. 

## Beneficios y Casos de Uso
**Ingeniería de Datos (ETL/ELT)**: Procesamiento y transformación de datos a escala.

**Ciencia de Datos e IA:** Desarrollo y despliegue de modelos de Machine Learning y GenAI.

**Analítica de Negocios (BI)**: Creación de tableros y análisis de datos en tiempo real.

**Integración en la Nube**: Disponible en Microsoft Azure, AWS, y GCP. 

## Aprender Databricks
Existe una Databricks Community Edition gratuita, ideal para practicar y se ofrecen cursos introductorios como el "Databricks Fundamentals" para entender la plataforma.
